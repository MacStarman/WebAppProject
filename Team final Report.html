Instructions on how to install and run the project

Steps to install

  1. Download WebAppProject.zip
  2. Extract WebAppProject.zip to new directory
  3. Open command prompt
  4. Navigate to director
  5. To run, use command "mvn spring-boot:run"
  6. Go to localhost:8080 in web browser to start.

Reflections on what we have learnt and what we would do differently

We started the project by dividing the workload and creating an excel spreadsheet so people could decide what task to do. This worked well at the start, however we should have made deadlines for tasks since they are so intertwined with other coding in the application. We also set up a github to share code which was instrumental in seeing how each other was doing and for putting the final project together. In reflection we should have split tasks equally for documentaion, as the orginal plan relied on other team members having their assigned sections completed, in order to work on their own agreed part. We focused on the backend for the majority of the project which meant we had to put together a template frontend page and linked everyones code to it. We also had to collate three different user controller java files together, whereby we learned that if we did the project in the future we would create a template html page and java controller files first to lessen the work need to bring everyones code together at the end.
